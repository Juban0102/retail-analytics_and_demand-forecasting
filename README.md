# Retail Sales Analytics and Demand Forecasting

## Overview

This project builds an end-to-end retail analytics platform that:

 - Stores retail transaction data in PostgreSQL.
 - Performs business analytics using SQL.
 - Forecasts future sales with machine learning.
 - Presents insights through an interactive Streamlit dashboard.

## Technologies
 - Python
 - PostgreSQL
 - Pandas
 - SQLAlchemy
 - Scikit-Learn
 - Prophet
 - Plotly
 - Streamlit
